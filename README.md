# Delo Task Manager System

##  Project Description

Delo Task Manager System is a web-based application built using Spring Boot.
It allows users to manage tasks efficiently by creating, viewing, updating, and deleting tasks through a simple interface.

---

##  Features

* Create new tasks
* View all tasks
* Update existing tasks
* Delete tasks
* Simple and user-friendly UI

---

##  Technologies Used

* Java
* Spring Boot
* Maven
* HTML / CSS
* Thymeleaf

---

##  How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/rayimbekovMederchik/delo-task-manager-system.git
cd delo-task-manager-system
```

### 2. Run the application

Using Maven:

```bash
./mvnw spring-boot:run
```

Or:

```bash
mvn spring-boot:run
```

### 3. Open in browser

```
http://localhost:8080
```
